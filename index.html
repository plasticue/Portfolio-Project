01. Movie-Fight - Web App

Browse your favorite movies and compare their stats

Movie Fight is a web app that uses IMDb API with two endpoints , one for movie search and one for movie details. Axios library is used to fetch data and CSS framework Bulma is used for easy styling. You can search and select two movies and displayed stats will be compared

02. Expense-Tracker - Web App

Track and calculate your expenses quickly with this web app built in vanilla JavaScript

The web application that allows you to quickly track and calculate your expenses with the option to add and remove items from a list with total balance. This app is built in vanilla javaScript.

03. Hangman - Web Browser Game

All time classic hangman game with SVG and vanilla JavaScript

I think that everyone is familiar with this game,guess letters until they hang you. one more project that I built in vanilla JavaScript and this time for animation I used SVG.

04. Breakout - Web Browser Game

Breakout game with canvas API and vanilla JavaScript

Classic breakout game built in vanilla JavaScript with a strong focus on the canvas. HTML canvas and javaScript canvas API is used. Now go break some bricks!

05. Speed-Typing - Web Browser game

Practice your speed typing on three difficulties

Speed-Typer is a web browser game where you practice your typing speed. Every time you type a word correctly your time left is incremented for the amount of second which depends on the difficulty on which you are playing. The game has the option to choose between difficulties easy, medium, and hard.
